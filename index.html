Tentang

Perkenalkan saya Ilham Sholeh lulusan Universitas Duta Bangsa Surakarta mendapatkan predikat cumlaude dengan nilai indeks prestasi kumulatif (IPK) 3,82 dari jurusan teknik informatika pada 4 Oktober 2023, skripsi yang saya buat untuk menyelesaikan program studi sastra satu teknik informatika tentang deteksi jumlah rak dari citra desain layout berbasis image processing menggunakan Metode YOLO dari perusahaan PT Adi Rak Minimarket Solo. Untuk melihat program yang saya buat anda bisa melihat pada github saya atau bisa hubungi saya https://github.com/SholehUyee untuk melakukan demo program, program deteksi berhasil dibuat yang dapat digunakan untuk mendeteksi jumlah rak dari gambar desain yg dimasukkan dan melakukan input dari ukuran ruangan serta terdapat fitur kalkulator untuk menghitung bagian dari rak minimarket.

saya pernah bekerja sebagai bagian admin dan bagian penjualan produk distributor rak dan ritel selama kurang lebih 2 tahun di perusahaan PT Adi Rak Minimarket Solo mempromosikan produk rak di media sosial, mengelola halaman perusahaan, pembuat iklan produk, menangani pembuatan desain layout untuk kebutuhan rak minimarket dari ukuran ruangan, pengiriman barang, maupun pemasangan produk rak Minimarket, dan sejenisnya. Saya pernah mengikuti kegiatan bootcamp internet of things yang diadakan kampus Universitas Duta Bangsa Surakarta untuk pembuatan suatu alat smart city yang diujikan untuk lomba tingkat universitas di Politeknik Solo, kami bekerja sama 4 orang dalam satu tim sehingga kami mendapatkan juara satu dalam perlombaan IOT dengan pembuatan alat Smart Parking sesuai dengan postingan saya. Saya pernah mengikuti kegiatan magang dari Universitas di bagian pembuatan content desain untuk kebutuhan media sosial di Duxeos Software House, untuk membuat beberapa macam konten baik desain untuk informasi loker, desain produk, maupun editing photo untuk di-posting di media sosial, serta melakukan pembuatan artikel untuk postingan dari setiap konten.

Saya memiliki kemampuan di bidang multimedia yang sewaktu sekolah mengambil jurusan multimedia di SMK Negeri 6 Sukoharjo, melakukan kegiatan Prakerin di Hero Jajar bagian desain grafis membuat beragam macam desain maupun membuat produk souvernir, melakukan kegiatan magang dalam pembuatan efek animated atau menggerakkan objek animasi 3d menggunakan software blender di GMC Animation yg berada di Surakarta saya amat tertarik dalam hal modeling pembuatan produk efek animasi 3d. produk terbaru yg saya buat ialah produk rak untuk kebutuhan pembuatan desain ruangan yg membutuhkan rak, saya membuat dengan refrensi langsung dari produknya.
